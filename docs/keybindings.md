
## zsh
### fzf-gems keybindings
ctrl-g ctrl-{f b t r h}

- C-g C-f => `git status`
- C-g C-b => `git branch`
- C-g C-t => `git tag`
- C-g C-r => `git remote`
- C-g C-h => `git log`

## emacs
### org-agenda
- C-a => `org-agenda`
- C-c C-s => `add schedule`
- C-c C-t => `toggle todo state`
- C-c C-x C-o => `clock in`
- C-c C-x C-o => `clock out`
